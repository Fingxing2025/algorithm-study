# 算法学习与国赛复习仓库

这个仓库用于整理个人算法学习资料，内容分为两部分：

- 国赛复习资料：偏向蓝桥杯国赛相关的复习笔记、训练路线和题目练习。
- C++ 模板库：按专题整理的单文件算法模板，方便查阅、复用和赛前快速回顾。

## 目录说明

### 国赛复习

主要包含以下内容：

- DP、图论、倍增法等专题复习笔记。
- 国赛冲刺策略、技术栈梳理、训练路线。
- 少量专题代码或题目实现，例如 P5012\_优化版.cpp。

### 模板库

模板库按常见算法专题划分，当前包含：

- 基础
- 二分
- 树状数组
- 数据结构
- 搜索
- 图论
- dp

大部分模板以单文件形式保存，适合：

- 赛前快速翻阅
- 做题时直接复制后按需修改
- 结合注释回顾算法思路和边界处理

## 使用方式

仓库中的大多数 C++ 文件都可以单独编译运行。

### Windows PowerShell

```powershell
g++ -std=c++17 -O2 "模板库\图论\LCA.cpp" -o lca
.\lca.exe
```

### 语法检查

如果只想检查模板是否能通过编译，可以使用：

```powershell
g++ -std=c++17 -fsyntax-only "模板库\图论\LCA.cpp"
```

## 仓库特点

- 以中文目录和文件名组织，便于按知识点检索。
- 以单文件模板为主，不依赖统一工程结构。
- 更强调竞赛场景下的实用性和复用效率。
