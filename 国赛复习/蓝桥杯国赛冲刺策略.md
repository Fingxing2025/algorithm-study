# 蓝桥杯 C/C++ 国赛冲刺策略

适用对象：

- 已拿到省一。
- 还有一个多月准备时间。
- 当前知识体系以现有模板库为主。

这份策略不是按 NOI / ICPC 顶级难度来配，而是按近几年蓝桥杯 C/C++ 国赛的实际结构来配。

---

## 1. 先判断比赛风格，而不是先刷高难专题

结合近几年公开题解和赛后复盘，蓝桥杯国赛的整体结构更接近下面这种分层：

1. 前 2 到 3 题：偏签到或结果填空，但容易卡细节、卡数据类型、卡读题。
2. 中间 3 到 4 题：真正拉开普通选手和省一等奖选手差距，常见是数学规律、构造、贪心、前缀和、双指针、BFS/最短路、基础 DP、基础图论。
3. 最后 2 到 3 题：明显上强度，往往允许暴力分、部分分，满分需要更强的数据结构、图论或状态设计。

所以你接下来一个多月最重要的目标，不是把所有高难模型都学完，而是做到：

1. 前 5 到 6 题尽量稳定拿满。
2. 第 7 到 8 题争取正解或高额部分分。
3. 第 9 到 10 题至少能快速写出一版有分暴力。

一句话概括：

> 蓝桥杯国赛更像“前中段稳定收分 + 后段快速抢分”，而不是“只靠最后两题神仙打架”。

---

## 2. 你现在最该调整的复习方向

你原来的知识库已经开始往“国赛级 DP 专题”扩了，这本身没有问题。

但如果目标明确是蓝桥杯国赛，而且只剩一个多月，那么必须做两件事：

1. 把基础与中档题型的稳定性放到最高优先级。
2. 把超重型专题的优先级下调，只保留最有可能直接转化成分数的部分。

### 2.1 当前应上调优先级的内容

这些是蓝桥杯国赛里更容易直接出分的：

1. 模拟与分类讨论
2. 前缀和、差分、双指针、滑窗
3. 排序、离散化、哈希统计
4. BFS / DFS / 状态最短路
5. 并查集、最短路、拓扑、最小生成树等基础图论
6. 数学：质数筛、因数分解、模运算、组合计数、gcd/lcm、构造找规律
7. 基础 DP：线性 DP、背包、简单计数、简单状态 DP
8. 暴力优化与部分分思维

### 2.2 当前应下调优先级的内容

这些并不是不重要，而是“一个多月内性价比不够高”：

1. Knuth 优化
2. 分治优化 DP
3. 插头 DP
4. 非常硬的轮廓线 / 状压细节题
5. 非常重的树上高级 DP 变形
6. 复杂概率图 DP

对蓝桥杯来说，你当然可以会这些，但它们不应该挤占前中段收分题型的训练时间。

---

## 3. 依据近几年国赛，题型优先级应这样排

### 第一优先级：必须稳定拿分

1. 模拟 / 构造
2. 数学规律题
3. 前缀和 / 差分 / 哈希统计
4. 双指针 / 贪心
5. BFS / 网格最短路 / 状态 BFS
6. 基础 DP

原因：

- 这些题出现频率高。
- 题面不一定吓人，但很考验读题和实现细节。
- 省一选手在国赛里最容易靠这里拉满基础盘。

### 第二优先级：决定你能否往前排冲

1. 并查集
2. 最短路
3. 拓扑 DP / DAG DP
4. 树形 DP 基础题
5. 数位 DP 基础题
6. 状态压缩 DP 基础题
7. Tarjan / 割边割点基础应用

原因：

- 这批题往往出现在中后段。
- 不一定是最顶，但写得快、写得稳的人会明显领先。

### 第三优先级：保留最低可用认知和暴力抢分能力

1. 区间 DP
2. 换根 DP
3. 树上背包
4. 概率期望 DP
5. 斜率优化 DP

要求不是“都打通”，而是：

- 见过
- 知道大概状态
- 能判断什么时候该直接放弃正解，先写暴力

---

## 4. 你的模板库要怎么用，才更符合蓝桥杯

你现在的模板库是个不错的起点，但蓝桥杯里模板的正确用法不是“背越多越好”，而是：

1. 每类题都要有一个最小模板。
2. 每个最小模板都要能手写出来。
3. 每份模板都要知道配什么题型关键词。

具体到现有库，建议这样用：

### 4.1 基础目录

重点复习：

- 高精度类
- 数学部分
- 日期模板
- int128 示例
- string 类整理

原因：

- 蓝桥杯很爱出“实现不难，但细节容易炸”的题。
- 数据范围和类型处理不稳，会白丢很多分。

### 4.2 搜索目录

重点复习：

- bfs二维网格最短路.cpp
- dfs 全排列.cpp

要补的意识：

- 状态 BFS
- 三维 visited
- 有限制条件的最短路

### 4.3 图论目录

重点复习：

- 并查集模板.cpp
- dijkstra+链式前向星..cpp
- 拓扑排序-kahn+邻接表.cpp
- kruskal最小生成树.cpp
- floyd+邻接矩阵.cpp

建议新增但先不重写：

- Tarjan 割边 / 割点基础模板

### 4.4 DP 目录

对蓝桥杯来说，优先级建议改成：

1. LIS / 编辑距离 / 基础计数 DP / 背包
2. 单调队列优化线性 DP
3. DAG DP / 树形 DP / 数位 DP
4. 状压 DP / 换根 DP
5. 斜率优化、轮廓线、概率期望只保留“认识”和最小骨架

## 5. 一个多月的最优时间分配

### 第 1 阶段：前 10 天，打稳基础盘

目标：

- 把蓝桥杯前 5 到 6 题常见模型全部稳定化。

每天建议：

1. 1 道模拟 / 构造
2. 1 道数学 / 数论 / 筛法
3. 1 道前缀和 / 双指针 / 差分 / 哈希统计
4. 1 道 BFS / 最短路 / 简单图论

这阶段最重要的不是题量，而是：

- 每题写完后，把“为什么一开始会卡住”补进文档。

### 第 2 阶段：中间 12 到 14 天，补中后段拿分题

目标：

- 把中档图论和中档 DP 补齐到“会判断、会写最小模板”。

建议顺序：

1. 并查集 / 最短路 / 拓扑 / MST
2. 树形 DP / DAG DP / 数位 DP
3. 状压 DP / 换根 DP
4. Tarjan 割边割点基础应用

### 第 3 阶段：最后 10 天，按比赛方式冲刺

目标：

- 训练蓝桥杯国赛最关键的东西：做题顺序、卡时策略、部分分意识。

建议：

1. 限时做整套国赛题或高质量模拟赛
2. 固定做题顺序
3. 强制练“最后两题 20 分暴力”

这阶段不建议再开新专题。

---

## 6. 比赛策略要改成“分数最大化”，不是“全题执念”

蓝桥杯国赛里，最容易把省一选手坑掉的不是不会，而是：

1. 前面写慢了
2. 中间题调细节调炸了
3. 后面没有及时写暴力

建议比赛策略：

### 6.1 理想顺序

1. 先秒前面结果填空 / 签到题
2. 再做实现中档题
3. 再做自己最熟的图论或 DP 题
4. 最后统一给尾题抢部分分

### 6.2 明确止损线

如果一道题：

- 20 分钟还没建模清楚
- 40 分钟还没接近可交版本

就必须开始考虑：

1. 有没有暴力分
2. 有没有小数据分
3. 有没有特殊性质分

### 6.3 蓝桥杯国赛的关键能力

不是“我会不会那道最难题”，而是：

1. 我能不能稳定拿下该拿的题
2. 我能不能快速把陌生题拆出暴力
3. 我能不能不在简单题上炸细节

---

## 7. 对 DP 部分的专门调整

如果你按原来的 DP 进阶路线全开，会偏重。

针对蓝桥杯，一个多月内建议这样调整：

### 必须打通

1. 线性 DP
2. 背包
3. 简单计数 DP
4. DAG DP
5. 树形 DP 基础
6. 数位 DP 基础

### 要会模板，但不必深挖

1. 状压 DP
2. 换根 DP
3. 区间 DP

### 只保留认知，不抢时间

1. 轮廓线 DP
2. 概率期望 DP
3. 斜率优化 DP
4. 分治优化 / Knuth / 插头 DP

原因很简单：

- 蓝桥杯国赛会出 DP，但整体不像 NOI 那样重型 DP 占主导。
- 你现在更需要的是“中档 DP 稳拿 + 能看出尾题暴力”。

---

## 8. 这一个月你最该补的三个非算法细节

### 8.1 类型安全

近几年题解复盘里，一个非常高频的丢分点就是：

- `int` 开小了
- `long long` 还不够
- 无符号、乘法溢出、平方溢出没有提前防

所以：

- `int128` 要熟
- `long long` 与 `unsigned long long` 的边界要清楚
- 平方、乘积、前缀和都要先看范围

### 8.2 细节实现

蓝桥杯很常见这种坑：

- 下标越界
- 前缀和边界
- 多组输入没清空
- 图数组没初始化
- BFS 状态没去重

### 8.3 暴力意识

最后几题就算不会正解，也要先写：

1. 小数据暴力
2. 特判
3. 特殊性质
4. 降一维 / 降状态后的次优做法

---

## 9. 一页总结

对你现在这个阶段，最优策略不是“把知识树无限扩展”，而是：

1. 把蓝桥杯中前段的高频题型做到非常稳。
2. 把图论和中档 DP 补到能拿中后段分。
3. 养成最后两题抢暴力分的固定习惯。

如果只用一句话概括：

> 蓝桥杯国赛一个多月冲刺的最优解，不是转型成 OI 重型选手，而是把“稳定 AC + 中档突破 + 尾题抢分”做成习惯。
