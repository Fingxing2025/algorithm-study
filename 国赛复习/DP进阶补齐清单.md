# DP 进阶补齐清单

这份清单是基于当前知识库做的缺口判断。

补充说明：

这份清单默认视角是“算法竞赛进阶 DP 体系”。
如果目标明确是蓝桥杯 C/C++ 国赛，请结合 蓝桥杯国赛冲刺策略.md 一起看，优先级要重新排序。

你现在的 DP 基础已经覆盖：

- 线性 DP
- 基础字符串 DP
- 基础计数 DP
- 背包全家桶中的常见变种
- 单调队列、滚动数组、二进制拆分等初阶优化

如果目标是国赛，这还不够。真正的缺口主要在：

1. 树上 DP
2. 区间 DP
3. 数位 DP
4. 状态压缩 DP
5. DAG DP / 拓扑 DP
6. 换根 DP

这六类是“必须补齐”的第一梯队。

第二梯队是：

1. 树上背包
2. 轮廓线 DP
3. 概率 / 期望 DP
4. 斜率优化 DP
5. 决策单调性优化 DP
6. 插头 DP 或更强的状态设计题

但如果只剩一个多月，且目标是蓝桥杯国赛，那么第二梯队里真正建议保留的只有：

1. 树上背包
2. 概率 / 期望 DP 的基础认知
3. 斜率优化 DP 的基础认知

轮廓线、插头、Knuth、分治优化这类更偏 OI / 省选风格的内容，不应抢走中前段高频题型的训练时间。

如果只从国赛实战收益看，我建议先按下面顺序补：

1. 树形 DP
2. 区间 DP
3. 数位 DP
4. 状压 DP
5. 换根 DP
6. DAG DP
7. 树上背包
8. 概率期望 DP
9. 斜率优化 / 决策单调性

## 1. 你当前最缺的六类模型

| 模型    | 为什么必须补                               | 当前模板库情况 | 建议优先级 |
| ------- | ------------------------------------------ | -------------- | ---------- |
| 树形 DP | 国赛高频，很多题本质是“子树信息合并”       | 缺             | 最高       |
| 区间 DP | 合并、括号化、环形链、染色类题必备         | 缺             | 最高       |
| 数位 DP | 统计区间内满足数位限制的个数几乎是固定考点 | 缺             | 最高       |
| 状压 DP | 小规模集合最优化、TSP、棋盘类常见          | 缺             | 最高       |
| 换根 DP | 树上全点答案、树上距离和、重心类很常见     | 缺             | 高         |
| DAG DP  | 图上最值、计数、路径类问题的基础形态       | 缺             | 高         |

一句话判断：

> 你现在的 DP 库已经够打提高组基础题，但还不够打国赛常见的树、图、状态压缩、数位和高阶优化题。

## 3. 每一类该怎么学

### 3.1 树形 DP

先学什么：

- 子树合并
- 选或不选当前点
- 子树内最优值

最小模板：

- 模板库/dp/进阶/树形dp.cpp

应该掌握的状态：

- `dp[u][0/1]`
- `sz[u]`
- 子树内统计信息

推荐例题：

1. P1352 没有上司的舞会
2. P2015 二叉苹果树
3. P2014 选课

最容易错：

- 树上转移顺序错，把父亲和儿子混在一起。
- 忘记“当前点选了，儿子就不能选”这类依赖关系。

### 3.2 换根 DP

先学什么：

- 先求一个根下的答案
- 再通过父子关系把答案从父亲推到儿子

最小模板：

- 模板库/dp/进阶/换根dp.cpp

应该掌握的状态：

- `sz[u]`
- `down[u]`
- `ans[u]`

推荐例题：

1. P3478 STA-Station
2. 树上距离和类题
3. 树上最远点距离类题

最容易错：

- 第二遍 DFS 的转移公式不会推。
- 不清楚“换根后有多少点离新根更近 / 更远”。

### 3.3 区间 DP

先学什么：

- 按区间长度递推
- 枚举最后一次合并 / 最后一个断点

最小模板：

- 模板库/dp/进阶/区间dp.cpp

应该掌握的状态：

- `dp[l][r]`
- 前缀和 `sum[i]`

推荐例题：

1. P1880 石子合并
2. P1063 能量项链
3. P4170 涂色

最容易错：

- 循环顺序不按区间长度。
- 环形区间不会转成链复制两倍处理。

### 3.4 数位 DP

先学什么：

- 按位 DFS
- `tight` 限制
- `started` 前导零状态
- 记忆化缓存

最小模板：

- 模板库/dp/进阶/数位dp.cpp

应该掌握的状态：

- `dfs(pos, state, tight, started)`

推荐例题：

1. P2657 Windy 数
2. P2602 数字计数
3. P4127 同类分布

最容易错：

- 前导零算不算进状态。
- `tight` 为真时不能记忆化全部状态。

### 3.5 状压 DP

先学什么：

- 用二进制集合表示“哪些点已经选过”
- 在集合上做转移

最小模板：

- 模板库/dp/进阶/状压dp.cpp

应该掌握的状态：

- `dp[mask][i]`

推荐例题：

1. P1433 吃奶酪
2. TSP 经典模型
3. 棋盘放置类状压题

最容易错：

- 集合是否包含当前点。
- 初始状态和终止状态设错。

### 3.6 DAG DP

先学什么：

- 拓扑排序
- DAG 上的最短路 / 最长路 / 路径计数

最小模板：

- 模板库/dp/进阶/DAGdp.cpp

应该掌握的状态：

- `dp[u]`
- 拓扑序推进

推荐例题：

1. P1807 最长路
2. P4017 最大食物链计数
3. DAG 上路径计数类题

最容易错：

- 图不保证连通时只从单源推，漏掉别的入度为 0 点。
- 最长路初始化成 0，导致不可达状态混入答案。

## 4. 第二梯队也要逐步补齐

这些不一定马上写进模板库，但你要有明确的学习路线。

### 4.1 树上背包

典型特征：

- 树结构
- 有容量 / 选点数限制
- 子树合并时像分组背包

推荐例题：

1. P2014 选课
2. 树上选点、保留若干边、选若干课程类题

### 4.2 轮廓线 / 棋盘状压

典型特征：

- 小宽度网格
- 行与行之间状态相关

推荐例题：

1. P1879 Corn Fields
2. P1896 互不侵犯

### 4.3 概率 / 期望 DP

典型特征：

- 每一步有若干概率转移
- 题目求期望步数、期望代价、期望收益

推荐例题：

1. P4316 绿豆蛙的归宿
2. 炸弹、抽卡、随机游走类期望题

### 4.4 斜率优化 DP

典型特征：

- 转移式长得像
- `dp[i] = a[i] + min/max(dp[j] + X(j) * Y(i) + Z(j))`

推荐例题：

1. P3195 玩具装箱
2. P3628 特别行动队

### 4.5 决策单调性优化

典型特征：

- 有断点 / 分割点
- 最优决策点随区间单调移动

推荐例题：

1. 四边形不等式类区间 DP
2. 分治优化 DP 经典题

## 5. 你现在最合理的补库顺序

如果改成“蓝桥杯国赛一个多月冲刺”的版本，更推荐下面这个顺序：

1. 树形 DP
2. DAG DP
3. 数位 DP
4. 换根 DP / 状压 DP 只做基础模板
5. 树上背包只做一轮基础题
6. 轮廓线、概率期望、斜率优化只保留认识和最小模板

如果要一起把知识库补得更像“国赛可用版”，建议分三轮：

### 第一轮：先补必会模板

1. 树形 DP
2. 换根 DP
3. 区间 DP
4. 数位 DP
5. 状压 DP
6. DAG DP

### 第二轮：补专题分支

1. 树上背包
2. 环形区间 DP
3. 棋盘状压 DP
4. 概率 / 期望 DP

### 第三轮：补国赛优化

1. 斜率优化 DP
2. 分治优化 DP
3. Knuth 优化 / 四边形不等式

## 6. 一页总结

你当前 DP 库的问题不是“基础不够”，而是“进阶块还没成体系”。

最该立刻补齐的是：

1. 树形 DP
2. 区间 DP
3. 数位 DP
4. 状压 DP
5. 换根 DP
6. DAG DP

如果这六类你都能写出基础模板，再往上加树上背包、概率期望、斜率优化，国赛里的 DP 才算进入比较完整的状态。
