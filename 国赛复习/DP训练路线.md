# DP 训练路线

这份文档的目标不是再解释一遍 DP，而是把现有知识库整理成“该先练什么、每类看哪个模板、先刷哪几题”的路线图。

注意：这份路线原本偏“泛国赛 DP 体系”。如果你的目标明确是蓝桥杯 C/C++ 国赛，而且只剩一个多月，请优先配合 蓝桥杯国赛冲刺策略.md 使用。

建议用法：

1. 先看这一类的最小模板。
2. 手写一遍状态和转移。
3. 按顺序刷 2 到 3 道题。
4. 最后回到复习文档，把这一类的易错点补成自己的笔记。

## 3. 第一梯队进阶：国赛必须补齐的六类

### 3.1 树形 DP

- 模板：模板库/dp/进阶/树形dp.cpp
- 建议顺序：先做选或不选，再做子树容量限制
- 例题：P1352 没有上司的舞会，P2015 二叉苹果树，P2014 选课

### 3.2 换根 DP

- 模板：模板库/dp/进阶/换根dp.cpp
- 建议顺序：先做树上距离和，再做树上全点答案
- 例题：P3478 STA-Station，P2986 Great Cow Gathering，树上距离和类题

### 3.3 区间 DP

- 模板：模板库/dp/进阶/区间dp.cpp
- 建议顺序：先做链上区间，再做环形区间
- 例题：P1880 石子合并，P1063 能量项链，P4170 涂色

### 3.4 数位 DP

- 模板：模板库/dp/进阶/数位dp.cpp
- 建议顺序：先做“禁某个数字”，再做“数位和 / 模限制”
- 例题：P2657 Windy 数，P2602 数字计数，P4127 同类分布

### 3.5 状压 DP

- 模板：模板库/dp/进阶/状压dp.cpp
- 建议顺序：先做 TSP，再做棋盘行状态压缩
- 例题：P1433 吃奶酪，P1896 互不侵犯，P2704 炮兵阵地

### 3.6 DAG DP

- 模板：模板库/dp/进阶/DAGdp.cpp
- 建议顺序：先最长路，再路径计数
- 例题：P1807 最长路，P4017 最大食物链计数，P1137 旅行计划

第一梯队过关标准：

- 每一类都能独立写出最小模板。
- 看到题能判断它是“树、区间、数位、集合、DAG”中的哪一类。

## 4. 第二梯队：国赛拉开差距的专题

对于蓝桥杯国赛，这一层的优先级要整体下调。

建议理解成：

1. 先认识模型。
2. 先会最小模板。
3. 如果前面的基础盘和第一梯队还不稳，就不要在这里投入太多整块时间。

### 4.1 树上背包

- 模板：模板库/dp/进阶/树上背包.cpp
- 适用题面：树结构 + 容量限制 + 子树合并像分组背包
- 建议顺序：先做依赖选课，再做保留若干边 / 选若干节点
- 例题：P2014 选课，P2015 二叉苹果树，P1273 有线电视网

### 4.2 轮廓线 DP

- 模板：模板库/dp/进阶/轮廓线dp.cpp
- 适用题面：网格很窄、行与行之间状态强相关、需要描述“当前轮廓”
- 建议顺序：先做行状态压缩过渡，再看真正轮廓线模板
- 例题：P1879 Corn Fields，P1896 互不侵犯，P2704 炮兵阵地

### 4.3 概率 / 期望 DP

- 模板：模板库/dp/进阶/概率期望dp.cpp
- 适用题面：每一步随机转移，求期望步数 / 代价 / 收益
- 建议顺序：先 DAG 上期望，再一般状态期望
- 例题：P4316 绿豆蛙的归宿，P1850 换教室，P1654 OSU!

### 4.4 斜率优化 DP

- 模板：模板库/dp/进阶/斜率优化dp.cpp
- 适用题面：转移式能整理成直线在若干查询点上的最值
- 建议顺序：先单调队列版凸包，再学可二分 / Li Chao Tree
- 例题：P3195 玩具装箱，P3628 特别行动队，P2120 仓库建设

第二梯队过关标准：

- 不只会背公式，而是知道“为什么这个题能转成背包 / 轮廓线 / 凸包”。
- 对蓝桥杯而言，做到“看得懂、写得出最小模板、知道什么时候不该硬啃”就够了。

## 5. 冲刺层：还没补模板，但必须知道方向

对蓝桥杯国赛，这一层暂时不建议在赛前一个多月里重投入。

除非你已经满足：

1. 前 6 题非常稳。
2. 图论基础题稳定。
3. 第一梯队 DP 题基本能独立写。

| 模型        | 为什么要学       | 推荐方向                   |
| ----------- | ---------------- | -------------------------- |
| 分治优化 DP | 国赛常见优化手段 | 从决策单调性和代价函数入手 |
| Knuth 优化  | 区间 DP 强优化   | 先理解四边形不等式         |
| 插头 DP     | 比轮廓线更进一步 | 放在棋盘专题最后学         |
| 概率图 DP   | 状态更复杂       | 在 DAG 期望后再补          |

---

## 6. 最实用的刷题顺序

如果按 4 周补库，我建议这样分：

### 第 1 周

1. 树形 DP
2. 区间 DP
3. 换根 DP

### 第 2 周

1. 数位 DP
2. 状压 DP
3. DAG DP

### 第 3 周

1. 树上背包
2. 概率 / 期望 DP
3. 轮廓线 / 棋盘状压过渡题

### 第 4 周

1. 斜率优化 DP
2. 把前面每类再挑一题综合题回刷
3. 补自己的错题模板

---

## 7. 一页总结

如果你要打国赛，DP 的目标不该只是“题会做”，而应该是：

1. 我能快速判断题型。
2. 我知道最小模板长什么样。
3. 我知道这一类题先刷哪三题最划算。

这份训练路线就是围绕这三件事整理的。
