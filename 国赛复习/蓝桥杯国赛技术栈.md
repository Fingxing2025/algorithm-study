# 蓝桥杯国赛技术栈

这份文档的目标不是列一份“竞赛万能知识树”，而是按蓝桥杯 C/C++ 国赛近几年的真实难度，把你需要掌握的技术栈拆成三层：

1. 初级：前 4 到 6 题和结果填空最常用，必须稳定拿分。
2. 中级：中段 2 到 3 题最常出现，决定你能不能从“能拿奖”走到“国奖靠前”。
3. 高级：最后 2 到 3 题与高额部分分常见，要求不是全精通，而是会识别、会最小模板、会抢分。

这就是我对蓝桥杯国赛难度结构的核心判断：

> 蓝桥杯国赛不是“全靠顶级难题分胜负”，而是“前中段稳定收分 + 中后段突破 + 尾题部分分”共同决定结果。

所以这份技术栈文档不会按 NOI / ICPC 顶配去排，而是按蓝桥杯国赛的性价比去排。

---

## 1. 初级技术栈：必须稳拿的基础盘

这一层对应蓝桥杯国赛前几题和很多填空题。

要求不是“看过”，而是：

1. 看到题能很快认出模型。
2. 10 到 20 分钟内能写出正确代码。
3. 不在边界、类型、读题上白丢分。

### 1.1 模拟与分类讨论

为什么重要：

- 蓝桥杯前面题非常爱考这个。
- 真正难点通常不是算法，而是读题、实现和细节不炸。

你要练到：

- 能把题目规则完整翻译成代码。
- 会主动检查边界、顺序、重复、循环停止条件。

洛谷高质量例题：

- P1563 玩具谜题
- P1328 生活大爆炸版石头剪刀布
- P2670 扫雷游戏

### 1.2 枚举、前缀和、差分

为什么重要：

- 蓝桥杯很喜欢“表面是实现，实质靠预处理”的题。
- 这类题最容易出现在前中段，分值稳定。

你要练到：

- 一维前缀和、二维前缀和会直接写。
- 差分能用于区间修改和统计。
- 能分辨“直接暴力”与“预处理后线性扫”的差别。

洛谷高质量例题：

- P3397 地毯
- P2367 语文成绩
- P1719 最大加权矩形

### 1.3 排序、双指针、简单贪心

为什么重要：

- 蓝桥杯中档题经常包一层故事壳，本质是排序后扫描。
- 这类题是中前段最容易稳定积累优势的地方。

你要练到：

- 排序后能想到“相邻关系”“左右夹逼”“局部最优”。
- 会先排序再判断是否能双指针或贪心。

洛谷高质量例题：

- P1094 纪念品分组
- P1102 A-B 数对
- P1678 烦恼的高考志愿

### 1.4 基础数学与数论

为什么重要：

- 蓝桥杯国赛的填空题和前几题经常有数学规律、质数、gcd/lcm、整除和构造。
- 这类题常常不是难在公式，而是难在能不能快速想到正确方向。

你要练到：

- 会 gcd / lcm、质因数分解、筛法。
- 看见整除、互质、倍数、因子会先往数论想。

洛谷高质量例题：

- P3383 【模板】线性筛素数
- P1029 [NOIP2001 普及组] 最大公约数和最小公倍数问题
- P3811 【模板】乘法逆元

### 1.5 基础字符串处理

为什么重要：

- 蓝桥杯很常把字符串当作实现和规律题载体。
- 前面题里经常夹着一个“写起来麻烦，但算法不重”的字符串题。

你要练到：

- 熟练处理子串、字符统计、大小写、逆序、格式化输出。
- 不在下标和边界上翻车。

洛谷高质量例题：

- P1308 统计单词数
- P1553 数字反转
- P5734 【深基6.例6】文字处理软件

## 2. 中级技术栈：决定你能不能冲到前排

这一层大体对应蓝桥杯国赛中段 2 到 3 题。

这些题通常不会像 NOI 那样极端重型，但已经不再是单纯实现题。

要求是：

1. 看到题能在较短时间内选对模型。
2. 最小模板能独立写出来。
3. 能稳定拿正解，至少也要能快速改出有分版本。

### 2.1 BFS 与最短路基础

为什么重要：

- 蓝桥杯里“状态最少步数”“网格最短路”“图上最优路径”非常高频。
- 这类题经常是中段分差题。

你要练到：

- 普通 BFS、网格 BFS、单源最短路能快速切换。
- 能分清什么时候是 BFS，什么时候必须上 Dijkstra。

洛谷高质量例题：

- P1443 马的遍历
- P1332 血色先锋队
- P3371 【模板】单源最短路径（弱化版）

### 2.2 并查集与连通性

为什么重要：

- 蓝桥杯国赛很喜欢把并查集藏在“关系合并”“联通块统计”“边加入后判断”里。
- 它也是最小生成树的基础部件。

你要练到：

- 并查集模板要写得非常稳。
- 知道它擅长“判断同组”，不擅长“求最短路”。

洛谷高质量例题：

- P3367 【模板】并查集
- P1525 关押罪犯
- P1111 修复公路

### 2.3 最小生成树

为什么重要：

- 蓝桥杯中段很适合出“把图连起来但代价最小”的题。
- Kruskal 难度适中，收益很高。

你要练到：

- 会区分最短路和最小生成树。
- 会写 Kruskal，知道 Prim 的适用场景。

洛谷高质量例题：

- P3366 【模板】最小生成树
- P1546 最短网络 Agri-Net
- P2872 [USACO07DEC] Building Roads S

### 2.4 拓扑排序与 DAG 思维

为什么重要：

- 蓝桥杯很喜欢“先做谁后做谁”的依赖题。
- 一旦把题识别成 DAG，很多题会立刻简单很多。

你要练到：

- 能判环。
- 能从拓扑排序自然过渡到 DAG DP。

洛谷高质量例题：

- P1113 杂务
- P1807 最长路
- P4017 最大食物链计数

### 2.5 基础 DP 核心模型

为什么重要：

- 蓝桥杯国赛会出 DP，但多数时候不是特别重型的 DP。
- 中档线性 DP、背包、树形基础 DP 的收益非常高。

你要练到：

- 线性 DP、背包、基础树形 DP 能独立写。
- 看题先写状态，再谈优化。

洛谷高质量例题：

- P1048 采药
- P2758 编辑距离
- P1352 没有上司的舞会

### 2.6 二分答案与判定型模型

为什么重要：

- 蓝桥杯中段很常见“答案有单调性”的题。
- 很多看起来难的题，只要敢先写判定函数，就会清晰很多。

你要练到：

- 先判断有没有单调性。
- 会写“枚举答案 + check”的标准结构。

洛谷高质量例题：

- P1182 数列分段 Section II
- P2678 跳石头
- P1873 砍树

## 3. 高级技术栈：后段题与高额部分分的主战场

这里的“高级”不是指省选 / NOI 最难那批，而是指：

1. 在蓝桥杯国赛后 2 到 3 题里更容易出现。
2. 写出正解能明显拉开差距。
3. 就算正解不会，知道模型也更容易抢到暴力分或性质分。

要求不是全部打穿，而是：

1. 会识别。
2. 会最小模板。
3. 知道什么时候该收手去写部分分。

### 3.1 状态图最短路 / 分层最短路

为什么重要：

- 蓝桥杯后段题经常不是纯图，而是“原图 + 额外状态”。
- 这类题往往能把最短路和 DP/限制条件结合起来。

你要练到：

- 会把“使用次数”“免费次数”“模式切换”并进状态。
- 会把问题改写成“状态点 + 转移边”的最短路。

洛谷高质量例题：

- P4568 [JLOI2011] 飞行路线
- P1948 [USACO08JAN] Telephone Lines S
- P2939 [USACO09FEB] Revamping Trails G

### 3.2 树形 DP / 换根 DP / DAG DP 进阶

为什么重要：

- 这是蓝桥杯国赛中后段最有可能出现、但又不至于重到离谱的一类高级模型。
- 比网络流、树链剖分更值得优先投资。

你要练到：

- 能认出“子树合并”“每个点都要求答案”“DAG 上计数/最值”这些信号。
- 最小模板能写，复杂变形至少能写第一步。

洛谷高质量例题：

- P3478 [POI2008] STA-Station
- P2015 二叉苹果树
- P1137 旅行计划

### 3.3 数位 DP

为什么重要：

- 蓝桥杯国赛很喜欢计数、限制、区间数字类问题。
- 数位 DP 是这类题里非常高性价比的高级模型。

你要练到：

- 会写 `solve(r) - solve(l-1)` 的套路。
- 会处理 `tight`、前导零、状态记忆。

洛谷高质量例题：

- P2657 Windy 数
- P2602 数字计数
- P4127 [AHOI2009] 同类分布

### 3.4 状压 DP

为什么重要：

- 蓝桥杯最后几题里，集合状态和小规模全排列型题很常见。
- 这类题数据范围通常就是在提醒你“别按普通 DP 写”。

你要练到：

- 能看懂 `mask` 的含义。
- 会用“已选集合 + 最后位置”或“行状态”设计状态。

洛谷高质量例题：

- P1433 吃奶酪
- P1896 互不侵犯
- P2704 炮兵阵地

### 3.5 区间 DP

为什么重要：

- 蓝桥杯中后段偶尔会出“合并顺序”“区间转移”“环形断环成链”这类题。
- 不算最高频，但一旦出现就很吃模型识别。

你要练到：

- 会按区间长度递推。
- 会写“枚举最后一次合并 / 分割点”的转移。

洛谷高质量例题：

- P1880 石子合并
- P1063 能量项链
- P4170 涂色

### 3.6 Tarjan / 缩点 / 割点割边

为什么重要：

- 蓝桥杯国赛后段有一定概率出现图结构性质题。
- 这类题不一定非要出得特别重，但会显著区分会图论的人和只会最短路的人。

你要练到：

- 知道 `dfn / low` 在干什么。
- 能识别题目是在问强连通、割点还是割边。

洛谷高质量例题：

- P3387 【模板】缩点
- P3388 【模板】割点（割顶）
- P2860 [USACO06JAN] Redundant Paths G

### 3.7 树上 LCA 与路径关系

为什么重要：

- 这类题在蓝桥杯里不一定比最短路更高频，但一旦出现，没准备就很容易直接卡死。
- 它属于“你不需要天天刷，但至少得会一版稳定模板”的类型。

你要练到：

- 会倍增 LCA。
- 知道它能服务于距离、祖先、路径交汇等问题。

洛谷高质量例题：

- P3379 【模板】最近公共祖先（LCA）
- P4281 [AHOI2008] 紧急集合 / 聚会
- P3884 [JLOI2009] 二叉树问题

## 4. 蓝桥杯国赛一个多月该怎么用这份技术栈

如果你现在距离国赛只剩一个多月，那最优做法不是从上到下平均刷，而是：

### 第一阶段：先把初级打到非常稳

目标：

- 前几题和填空题尽量不失误。
- 模拟、前缀和、排序、基础数学这些要做到“看见就会”。

### 第二阶段：中级技术栈做成主力得分点

目标：

- BFS / 最短路 / 并查集 / MST / 拓扑 / 基础 DP 要能稳定拿下。
- 这是最容易决定国奖档位的一层。

### 第三阶段：高级技术栈只抓高性价比块

优先顺序建议：

1. 状态图最短路
2. 树形 / DAG / 换根 DP
3. 数位 DP
4. Tarjan
5. 状压 DP
6. 区间 DP
7. LCA

原因：

- 这些更像蓝桥杯国赛后段题真正会用到的高级块。
- 比网络流、后缀自动机、插头 DP、分治优化、Knuth 优化更适合一个月内投入。

---

## 5. 当前不建议重投入的技术栈

如果你的目标是蓝桥杯国赛一个多月冲刺，下面这些不应该抢走太多时间：

1. 网络流
2. 后缀自动机 / 后缀数组
3. 树链剖分
4. 李超线段树
5. Knuth 优化 / 分治优化 DP
6. 插头 DP
7. 复杂计算几何

不是说这些没价值，而是：

- 它们更像更高层竞赛体系里的深水区。
- 在蓝桥杯国赛的一月窗口里，收益通常不如把中级技术栈练稳，再把几个高性价比高级块补齐。

---

## 6. 一页总结

如果把蓝桥杯国赛技术栈压缩成一句话，就是：

1. 初级技术栈负责稳住前中段基础分。
2. 中级技术栈负责打穿真正拉开差距的主战场。
3. 高级技术栈负责后段突破和部分分抢分。

真正适合蓝桥杯国赛的复习方式不是“学得越深越好”，而是：

> 按蓝桥杯的出题结构，把最常考、最值钱、最容易转化成比赛分数的技术栈优先练熟。
